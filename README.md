# Thren

A tabletop RPG system and setting.

## Repository Structure

```
/
├── tools/              # Development and play tools
│   └── inventory-sim/  # Inventory & tile mechanics simulator
│
├── design/             # Design process materials
│   ├── notes/          # Design decisions, brainstorming
│   ├── inspiration/    # Reference links, mood boards
│   └── references/     # Raw reference material
│
├── rules/              # The game rules
│
└── setting/            # World, lore, factions
```

## Tools

### Inventory Simulator

A web-based tool for testing the inventory and tile mechanics.

```bash
cd tools/inventory-sim
npm install
npm run dev
```
